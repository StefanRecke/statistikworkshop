# Statistikworkshop (DG PARO) — Kurswebsite

Begleitmaterial zum Statistikworkshop: praxisnaher Einstieg in Statistik und
reproduzierbare Auswertungen mit R.

Die Website liegt fertig gerendert im Ordner `docs/` und wird über GitHub Pages
veröffentlicht. Die im Kurs verwendeten Datensätze stehen auf der Startseite
der Website zum Download bereit (Ordner `docs/Daten/`).

Erstellt mit [Quarto](https://quarto.org) und R.
